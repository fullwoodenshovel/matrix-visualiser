# Matrix Visualiser v0.1.1

## What's new
This is stuff important for the end user, basically the technical changes, but summarised and less important changes ignored.
These are the new features:

- Added a [website](https://www.fullw.me/matrix)
- Added support for phone
- Improved README
- Optimisations, bug fixes and readability improvements


## Technical changes
These are all of the important new commits from this release.
This may include technical changes, architectural changes, bug fixes, etc:

- Update README with latest release link, fixed key command, instructions, example and contributions subsection
- Added support for touchscreen / phone
- Added support for porting to web
- Made parser recognise when expression should end and fixed non-unix ui
- Optimised background draws
- Optimised unnecessary call to for_each
- Colour changes, readability and flashes changes, and added case insensitivity for keywords
- Removed comments that should be issues


## Downloads

### Windows
- **matrix-visualiser-v0.1.1-windows-x86_64.zip**
  1. Extract the ZIP
  2. Open the 'matrix-visualiser' folder
  3. Double-click matrix-visualiser.exe

### Linux
- **matrix-visualiser-v0.1.1-linux-x86_64.zip**
  1. Extract the ZIP
  2. Open the 'matrix-visualiser' folder in terminal
  3. Run: `./matrix-visualiser`

### macOS
- **matrix-visualiser-v0.1.1-macos.zip**
  1. Extract the ZIP
  2. Open the 'matrix-visualiser' folder
  3. Right-click 'matrix-visualiser' and Open (first time only)
  4. Or run in terminal: `./matrix-visualiser`

### Website
Just visit [www.fullw.me/matrix](https://www.fullw.me/matrix) and get started!
